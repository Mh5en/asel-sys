# نظام إدارة المخزون والمحاسبة - شركة أسيل

نظام سطح مكتب مبني باستخدام Electron لإدارة المخزون والمحاسبة.

## المتطلبات

- Node.js (الإصدار 16 أو أحدث)
- npm

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm start
```

## بيانات الدخول الافتراضية (للتجربة)

- اسم المستخدم: `admin`
- كلمة المرور: `admin`

أو

- اسم المستخدم: `user`
- كلمة المرور: `1234`

## الملفات

- `main.js` - العملية الرئيسية لـ Electron
- `preload.js` - النص البرمجي preload
- `login.html` - صفحة تسجيل الدخول
- `styles/login.css` - تنسيقات صفحة تسجيل الدخول
- `scripts/login.js` - منطق تسجيل الدخول
